# Metodebeskrivelser

Denne mappe indeholder nøgterne og lægmandsvenlige beskrivelser
af de former for alternativ behandling, som systemet forholder sig til.

Formålet med metodebeskrivelserne er at:
- give overblik
- skabe begrebsafklaring
- reducere forvirring og forhåbninger

Beskrivelserne:
- er ikke anbefalinger
- er ikke behandling
- indeholder ingen garantier
- afspejler variation og uenighed

Metodebeskrivelserne fungerer som referencegrundlag
for både brugere og systemets svar.
