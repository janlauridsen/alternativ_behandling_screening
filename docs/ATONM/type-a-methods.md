# ATONM – Type A Method Classification (Normative)

**Status:** Normativ, låst

Dette dokument fastlægger de bindende **Type A-kendetegn** for udvalgte metoder i ATONM. Kendetegnene er beskrivende, ikke normerende, og anvendes udelukkende til orienterende præsentation efter afsluttet ATONM-indsnævring.

---

## Type A-dimensioner (låst)

1. **Oplevelsesdomæne**: kropsligt · mentalt · blandet · fortolkende/abstrakt
2. **Interaktionsform**: primært passiv · primært aktiv · dialogbaseret · kombineret
3. **Grad af guidning**: selvstyret · delt ansvar · primært behandlerstyret
4. **Abstraktionsniveau**: overvejende konkret · blandet · overvejende fortolkende
5. **Struktureringsgrad**: fast · delvist struktureret · åben
6. **Afhængighed af udøver**: lav · middel · høj
7. **Temporal struktur**: afgrænset forløb · gentagende over tid · løbende praksis
8. **Kropskontakt**: ingen · let · direkte
